Proof of Consumer Consent for Kumbu Connect SMS Messaging

This page documents the opt-in workflow used by Kumbu Connect for its Community Feedback SMS Program on toll-free number (833) 632-7146. The consumer opt-in method is a consumer-initiated text message using a participating CBO keyword.

Program: Community Feedback SMS Opt-In Type: Via Text Message Types: Surveys, follow-up questions, check-ins
The consumer must take a clear affirmative action to opt in. Consent is not bundled into a privacy policy, terms of service, or any unrelated acceptance flow.

How Consent Is Collected

1
The consumer receives a CBO keyword.

Kumbu Connect works with participating community-based organizations. Each participating CBO has a keyword such as KIWA.

2
The consumer sends the keyword by SMS.

The consumer texts the keyword from their own mobile phone to (833) 632-7146. This inbound text message is the affirmative opt-in action.

3
Kumbu Connect records the opt-in event.

The system stores the phone number, the keyword used to enroll, the timestamp of the inbound message, and the subscriber status as proof of consent.

4
The feedback workflow begins only after that opt-in.

The subscriber then receives community feedback prompts, optional follow-up questions, and future check-in messages unless they opt out.

Consent is never inferred from a prior relationship, prior service use, or acceptance of another document. The user must send the keyword themselves to start the SMS program.

Exact Opt-In Language Presented to Consumers

This is the disclosure Kumbu Connect uses with the keyword-based sign-up instruction:

Consumer Sign-Up Prompt

Text your participating CBO keyword to (833) 632-7146 to join Kumbu Connect Community Feedback. By texting the keyword, you agree to receive SMS messages from Kumbu Connect about community feedback surveys, follow-up questions, and periodic check-ins related to that CBO. Message frequency varies. Message and data rates may apply. Reply STOP to opt out. Reply HELP for help. Consent is not a condition of purchase.

This language explicitly references SMS messages, explains what messages the user will receive, and tells the user exactly how to stop or request help.

Representative Message Samples

First Program Message After Opt-In

This is the first automated message sent after the consumer texts the keyword:

Thank you for joining KIWA feedback. Reply with a number from 1 to 10 to rate how KIWA has helped your community.

Opt-Out Confirmation

You have been unsubscribed from Kumbu Connect community feedback messages. Reply HELP for instructions.

Help / Enrollment Guidance

Reply with your CBO keyword to begin feedback. Example keywords: KIWA.

Message Types and Consumer Expectations

Initial Session

After opt-in, Kumbu Connect sends a rating prompt, a follow-up count question, and an optional story prompt.

Future Check-Ins

Subscribers may later receive additional check-in messages about the same community feedback program if they remain subscribed.

Frequency

Message frequency varies. A typical feedback session is short and usually contains 2 to 5 messages.

Support

Consumers can reply HELP for enrollment guidance or contact john@kijenzi.com.

Opt-Out and Help

  • Consumers can reply STOP, END, QUIT, CANCEL, or UNSUBSCRIBE to opt out.
  • Consumers can reply HELP or INFO to receive instructions.
  • Kumbu Connect honors opt-out requests by marking the subscriber as opted out and stopping future messages.

Recordkeeping

Kumbu Connect retains the following opt-in records for each subscriber:

  • The mobile phone number that initiated the SMS conversation
  • The CBO keyword used to enroll
  • The timestamp of the inbound keyword message
  • The subscriber's current status, including active or opted out
  • The stored message transcript for the feedback session

These records document the user's affirmative opt-in action and can be used to verify that consent was collected before follow-up messaging began.
